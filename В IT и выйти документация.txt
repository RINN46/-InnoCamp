1. Регистрация
    * Endpoint: POST/users/register

    
    Request:
    json
    {
    "username": "string",
    "email": "string",
    "password": "string"
    }

    Response:
    json
    {
    "success": "bool",
    "user_id": "int",
    "message": "string"
    }

2. Вход в аккаунт
    Endpoint: POST /users/login

    Request:
    json
    {
    "login": "string", // email или username
    "password": "string"
    }

    Response:
    json
    {
    "success": "bool",
    "user_id": "int",
    "token": "string"
    }

3. Опрос
    Endpoint: POST /users/survey

    Request:
    json
    {
    "user_id": "int",
    "answers": [
        {
        "question_id": "int",
        "answer": "string"
        }
    ]
    }

    Response:
    json
    {
    "success": "bool",
    "survey_completed": "bool"
    }\

4. Анализ от ИИ, который не показывается
    
    Запрашиваемые данные:
    - массив записей, которые содержат айди вопросов

    Return:
    - Массив данных о товарах


5. Лента товаров 
    Endpoint: POST/users/items

    Запрашиваемые данные:
    - Массив данных о товарах от ИИ

    Rеturn:
    - Фото товара
    - Описание товара
    - id товара
    -пользователь лайкнул  True/False
6. Избранное

    Endpoint: POST /users/favorites

    Request:
    json
    {
    "user_id": "int",
    "item_id": "int",
    "action": "string" // "like" или "unlike"
    }

    Response:
    json
    {
    "success": "bool"
    }

7.Цены
    Endpoint:GET/user/prises

    Запрашиваемые данные:
    - id пользователя
    - есть ли подписка True/False

    Return:
    если подписки нет:
    -текст описанием подписки
    -предложить купить подписку
    если она есть:
    -предложить продлить подписку
8. профиль пользователя:
    Endpoint: GET/user/profile

    Request Headers:
    Authorization: Bearer <token>

    Response:
    json
    {
    "username": "string",
    "avatar": "string|null",
    "favorites_count": "int",
    "subscription_active": "bool"
    }
